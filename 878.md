# 初始化SDK

在程序的入口处设置如下：

```
public class TestApplication extends Application {
    private static final String TAG = TestApplication.class.getSimpleName();

    final String APPID = "S4EBUTASGJ";
    final String APPKEY = "GEIHI32WGJREIO";

    @Override
    public void onCreate() {
        super.onCreate();
        TtCloudManager.init(this, APPID, APPKEY);
    }
}
```
